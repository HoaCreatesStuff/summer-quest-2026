NYC Summer Birthday
## Editorial Icon System v3.0

## Vision

Create a cohesive collection of premium editorial icons representing memorable New York summer adventures.

These icons are designed primarily for a **5×5 mobile scorecard**, where they will usually appear between **48–64 px**.

Every icon should communicate instantly.

This is **not** an illustration system.

This is an **icon system with personality.**

---

# Style Direction

Use the approved **Direction B** visual language.

Do not explore other rendering styles.

The final icons should feel like:

- Premium enamel pins
- Boutique stationery
- Modern editorial graphics
- Retro travel iconography
- Handcrafted collectible stickers

Avoid:

- Product rendering
- Photorealism
- Clip art
- Emoji
- Kawaii
- Children's book illustration

---

# Design Philosophy

**Personality comes from silhouette.**

Every icon should still be recognizable if viewed as a single flat black shape.

Do not rely on:

- tiny accessories
- hidden Easter eggs
- intricate interior detail
- realistic rendering
- subtle texture
- tiny visual jokes

Instead, create personality through:

- exaggerated proportions
- asymmetry
- posture
- oversized graphic shapes
- unexpected silhouettes

If someone has to zoom in to appreciate the joke, simplify it.

---

# Design Priorities

Always make decisions in this order:

1. Readability
2. Silhouette
3. Personality
4. Shape
5. Color
6. Texture
7. Detail

Never sacrifice a higher priority for a lower one.

---

# Silhouette First

The silhouette is the primary design tool.

Every icon should be recognizable:

- at 48 px
- without color
- without texture
- without interior detail

Imagine every icon filled completely black.

If you cannot identify it immediately, redesign it.

---

# Simplicity Rules

Push the approved Direction B style another **15–20% simpler.**

Remove:

- tiny patterns
- realistic folds
- unnecessary highlights
- excessive shading
- dense texture
- decorative clutter
- secondary objects

Use:

- fewer shapes
- larger shapes
- cleaner geometry
- stronger silhouettes

If a detail disappears at 48 px, remove it.

---

# Shape Language

Use:

- chunky proportions
- rounded geometry
- soft corners
- bold graphic shapes
- clean negative space
- slightly exaggerated forms

Avoid:

- thin lines
- realistic anatomy
- intricate perspective
- delicate shapes
- sharp technical geometry

Every icon should feel intentionally designed rather than realistically drawn.

---

# Composition

One centered hero object.

No scenes.

No environments.

No backgrounds.

No decorative borders.

No multiple focal points.

The icon should occupy roughly **75–85%** of the available canvas.

---

# Personality System

Every icon receives **one defining visual joke**.

The joke should be obvious even at 48 px.

Good examples:

- Cross-eyed pigeon
- Corgi viewed from behind
- Giant floppy bow on a boot
- Buildings leaning together like they're gossiping
- Disco ball hanging slightly crooked
- Human pyramid with the bottom person visibly squished

Bad examples:

- Tiny hidden heart
- Small skyline reflection
- Decorative miniature accessories
- Details only visible when zoomed in

One memorable idea is stronger than five subtle ones.

---

# Detail Rules

Large shapes only.

No micro-details.

Every icon should use the fewest possible shapes while remaining recognizable.

Ask:

**Can this detail be removed?**

If yes, remove it.

Prefer:

- one oversized bow
- one bold wave
- one dramatic feather
- one thick handle
- one giant ribbon
- one strong fold

Avoid:

- intricate basket weave
- detailed windows
- realistic feathers
- complex reflections
- tiny paint splatters
- subtle wrinkles
- decorative stitching

---

# Color

Use only the approved project palette.

Maximum:

- 3–5 colors per icon

Use:

- one dominant color
- one secondary color
- one dark separation color
- one optional accent

Avoid:

- gradients
- realistic lighting
- metallic rendering
- rainbow palettes
- excessive highlights
- pure black
- pure white

---

# Texture

Texture is secondary.

It should support warmth without reducing clarity.

Use:

- subtle printed grain
- light screen-print texture
- slightly imperfect edges

Avoid:

- painterly brush strokes
- heavy paper texture
- dense noise
- realistic shading

If texture hurts readability, remove it.

---

# Lighting

Keep icons flat.

Do not simulate realistic light.

Only use a single darker shape if needed to separate forms.

No:

- glossy highlights
- rendered shadows
- 3D effects
- photorealistic volume

---

# Optical Weight

Every icon should feel equally large.

Do not simply scale every object equally.

Adjust visually.

For example:

- Boots may be taller.
- Birds may need larger bodies.
- Wide baskets may need more side padding.
- Human figures should use thicker limbs.

The entire collection should feel balanced when viewed together.

---

# No Repeated Hero Objects

Each quest owns a unique silhouette.

Do not reuse hero objects across the collection.

Every icon should immediately distinguish itself from every other icon.

---

# Scale Test

Every icon must be reviewed:

1. Large
2. At 48 px
3. Inside the real scorecard

Approval should be based on the 48 px version.

If it only looks good large, redesign it.

---

# Technical Output

Transparent PNG

Square canvas

Centered composition

No cropping

Consistent padding

High-resolution master

Clean alpha transparency

---

# Mental Model

Design every icon as if it needs to be cut from colored vinyl using a Cricut machine.

If the silhouette is too complicated to cut cleanly, it is too complicated for this system.

Bold shapes.

Clean curves.

Strong negative space.

Minimal interior cuts.

---

# Final Checklist

✓ Instantly recognizable

✓ Strong silhouette

✓ Reads at 48 px

✓ One hero object

✓ One memorable personality trait

✓ No micro-details

✓ Consistent visual weight

✓ Flat editorial style

✓ Uses approved palette

✓ Subtle texture only

✓ Belongs to the same icon family

---

# Goal

The final collection should feel like a cohesive set of **collectible editorial icons** that are immediately recognizable, playful, handcrafted, and unmistakably New York.

Someone should be able to identify each icon in under one second, even at mobile size.
